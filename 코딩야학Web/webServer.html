WEB Server 체험

웹서버 Bitnami MAMP 설치
https://bitnami.com/stack/mamp/installer

다양한 웹 서버중에 아파치(Apache)를 사용하기로 한다. 맥에는 아파치가 이미 설치가 되있지만 아파치를 실행하고 설정을 변경하는 등의 동작은 실습하는데 추후 어려움이 예상되어 Bitnami MAMP를 별도로 설치한다.
MAMP(Mac Apache MySqul PHP)란 아파치를 직접 설치하는 어려움을 해결해주는 프로그램이다. MAMP는 자동으로 아파치가 설치가된다.
Application/mampstack-apache2-htdocs 내의 파일을 삭제하고 내가 만들었던 web의 html파일들로 대치하면 나의 index.html 페이지 확인이 가능.

http://127.0.0.1:8080/index.html
127.0.0.1 -> IP Address : 인터넷에 연결된 각각의 컴퓨터의 주소
8080 -> PORT : 하나의 컴퓨터에 서버가 두개일 때 구분을 하기 위해 붙여준다. 8080이 붙으면 설치한 웹서버에 접속, 붙지않으면 원래 서버에서 접속된다.

통신

제어판-네트워크-상태-IP주소를 휴대폰을 통해 웹페이지를 확인해볼 수 있다.(단, 같은 네트워크 환경이어야 함)
주소 -> http://IP주소:8080/index.html
